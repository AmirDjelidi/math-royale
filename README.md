# Math Royale — Duel de Fonctions

Battle royale 50 vs 50 en temps réel, façon *Graphwar* : deux escouades pilotées par IA s'affrontent à coups de fonctions mathématiques. Chaque boule recharge puis tire sa propre courbe `f(x)` ; toucher un adversaire (ou souffler un astéroïde à proximité) l'élimine.

Les joueurs progressent d'une partie à l'autre (XP sauvegardée dans le navigateur) et débloquent des familles de fonctions plus complexes : droite → sinusoïde → parabole → gaussienne → double sinus → marche tanh. Chaque match démarre avec au moins 2 joueurs, complétés par des bots calés sur leur niveau.

Un seul fichier HTML autonome, sans dépendance ni backend.

## Jouer

Ouvrir [`index.html`](index.html) dans un navigateur, ou la version en ligne une fois publiée sur GitHub Pages.
